#HOSPITAL SYSTEM - PROVA SOLUÇÕES DE SOFTWARE PARA NEGÓCIOS


*Nesse projeto, fiz um backend para um hospital, onde os pacientes podem agendar suas consultas mais fácil e também os médicos administrarem sua agenda.*

END-POINTS PRINCIPAIS=
- Cadastrar
- Login
- Atualizar
- Listar

REGRAS PRINCIPAIS= 
- Somente os pacientes ativos podem realizar agendamentos
- Somente os médicos ativos podem receber agendamentos
- Não pode ter agendamento no mesmo horario para o mesmo médico
- Ao desativar tanto o Paciente, tanto o Médico os Agendamento dos respectivos também sao cancelados
